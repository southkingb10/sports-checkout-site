# Ur Image Studios — Dynamic Sports Ordering Platform

This repository contains the front-end and serverless code for Ur Image Studios’ dynamic sports photography ordering system.  
It includes athlete intake pages, gallery ordering, dynamic cart totals, and Stripe PaymentIntent integration deployed through Netlify Functions.

## Features
- Athlete intake workflow
- Gallery image selection + add-on ordering
- Real-time package/add-on price calculations
- Dynamic Stripe checkout using Payment Intents
- Serverless backend powered by Netlify Functions
- Mobile-first responsive design
